Architecture distribuée — un OS que je modifie et pilote en entier

Origine : migration complète d'un orchestrateur Windows (JARVIS v12.4) vers une infrastructure Linux native, reconstruite pour être souveraine, modifiable de bout en bout et pilotable à distance. Chaque machine du cluster est à la fois nœud de calcul, relais et poste de commande : depuis n'importe laquelle, un agent ou une IA peut agir sur toutes les autres — elles se réveillent, se partagent et se réparent entre elles.

M1 — Leader (6 GPU)

Nœud principal d'inférence : qwen3.5, deepseek-r1, distillations Claude Opus, embeddings. Orchestration des agents et du routage LLM.

Ubuntu6 GPULM Studio

M2 — Reasoning

Nœud de raisonnement : deepseek-r1, qwen3.5-35b. Failover et délestage de charge depuis M1.

Ubuntureasoning

M3 / M4 — Backup & Windows

M3 : Ollama backup (gemma3, deepseek-r1). M4 : nœud Windows 11 d'intégration (LM Studio GUI).

OllamaWindows

M5 — Interface / Relais

Nœud d'interface CPU léger : relais Ollama, point d'entrée, miroir de sauvegarde.

relaisbackup
Partage complet de fichiers — maillage SSHFS bidirectionnel (M1↔M2↔M5) : chaque machine monte le système de fichiers des autres, accès admin, persistant au login. Documents accessibles partout.
Action à distance par agent/IA — dispatcher LLM unifié + MCP : une requête sur n'importe quel nœud route vers le meilleur backend (local GPU, cluster, agents OpenClaw) et exécute des actions système distantes.
Fédération & auto-réveil — Wake-on-LAN + maillage AnyDesk (auto-accept, écran noir privé) : toute machine peut booter/réveiller une autre, prendre la main, relancer un service.
Auto-réparation cluster — timer self-heal : détecte services en échec, montages morts, pairs hors-ligne (réveil WoL) et rafraîchit le registre de compétences cross-machine.
Skills cross-machine — bibliothèque de compétences/agents synchronisée via le mesh : M1 (38) · M2 (17) · M5 (14), routage SSH GitOps source-of-truth.
Tout modifiable — du kernel et de systemd jusqu'aux agents et aux sites : configuration sous contrôle de version par machine (dépôts privés M1/M2/M3/M4/M5), reproductible.
